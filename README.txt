PDFsummary Created by Miguel Zavala


INSTRUCTIONS:

1) Select the desired PDF 
2) Click "Retreive PDF" and wait until "100% complete" displays
3) Within Page# Entry, type in desired page number to summarize/read 
	OR
   Type in "all" to retrieve all summary/information 
4) Check "Summary?" to get a summary or Off to only read the desired page/full article
5) Within #Summary words Entry, type in desired number of words to retrieve from page if summary
6) Click "READ"
